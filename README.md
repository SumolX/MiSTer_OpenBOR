# MiSTer_OpenBOR
A port of OpenBOR v3.1 for the MiSTer platform.

Installer:
     
      Install_OpenBOR.sh

Modules:

      /media/fat/OpenBOR/Paks

Execute:

      MiSTer main menu
      Select scripts
      Select OpenBOR.sh

Update Controls:

      Select Module (PAK File)
      Press Start
      Options
      Control Options...
      Setup Player 1...
      Assign Buttons and select OK
      Repeat for other Players/Gamepads
